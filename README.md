## 📌 Sobre o Projeto

O **My Journey** é um site desenvolvido como projeto prático durante o programa **Entra21**. A aplicação tem como objetivo registrar, estruturar e exibir a minha jornada de aprendizado, evolução técnica e projetos desenvolvidos ao longo do programa.

Ele funciona como um portfólio dinâmico e um diário de bordo da minha transição e crescimento no universo da tecnologia.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais do desenvolvimento Web:

*   **HTML5:** Estruturação semântica das páginas.
*   **CSS3:** Estilização, layout e responsividade.
*   **JavaScript:** Lógica, interatividade e dinamismo do site.
*   **Git & GitHub:** Versionamento de código e hospedagem do repositório.

---

## 📂 Estrutura do Repositório

A organização dos arquivos do projeto segue o padrão de desenvolvimento web:

*   `/css`: Arquivos de estilização do site.
*   `/img`: Imagens e assets visuais utilizados.
*   `/js`: Scripts e funções em JavaScript.
*   `index.html`: Página principal e ponto de entrada da aplicação.

---

## 🚀 Funcionalidades Principais

*   📱 **Design Responsivo:** Interface adaptável para dispositivos móveis, tablets e desktops.
*   📅 **Linha do Tempo (Timeline):** Visualização cronológica dos módulos aprendidos no Entra21.
*   📂 **Galeria de Projetos:** Espaço dedicado para exibir códigos criados durante o programa.
